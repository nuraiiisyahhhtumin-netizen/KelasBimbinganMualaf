## 🚀 Cara Deploy ke GitHub Pages

### Langkah 1 — Buat Repository
1. Log masuk [github.com](https://github.com)
2. Klik **New repository**
3. Nama repo: `emualaf-hcf-johor`
4. Set kepada **Public**
5. Klik **Create repository**

### Langkah 2 — Upload Fail
1. Dalam repo baharu, klik **Add file → Upload files**
2. Drag & drop kedua-dua fail:
   - `index.html`
   - `README.md`
3. Klik **Commit changes**

### Langkah 3 — Aktifkan GitHub Pages
1. Pergi ke **Settings → Pages**
2. Di bawah **Source**, pilih `Deploy from a branch`
3. Branch: `main` / folder: `/ (root)`
4. Klik **Save**
5. Tunggu 1–2 minit, laman akan aktif di:  
   `https://<username>.github.io/emualaf-hcf-johor`
